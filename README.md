# Mokesano Theme

News theme, simple, powerful and efficient WordPress theme for news and blogging sites.

## Requirements

- **WordPress:** At least version 6.8.5
- **Tested up to:** WordPress 6.8.5
- **PHP:** Requires version 8.0 or higher

## Features

- Custom Background
- Custom Logo
- Custom Menu
- Featured Images
- Threaded Comments
- Translation Ready

## Installation

1. In your admin panel, go to **Appearance > Themes** and click the **Add New** button.
2. Click **Upload Theme** and **Choose File**, then select the theme's `.zip` file. Click **Install Now**.
3. Click **Activate** to use your new theme right away.

### Alternative Installation Method

1. Download the theme zip file.
2. Unzip the folder and upload it to `/wp-content/themes/` directory.
3. In your admin panel, go to **Appearance > Themes** and click **Activate** on Sangia.org theme.

## Frequently Asked Questions

### Does this theme support any plugins?

Yes! Sangia.org includes support for:
- **WooCommerce** - for e-commerce functionality
- **Jetpack Infinite Scroll** - for seamless content loading

### Is this theme translation ready?

Yes, the theme is translation-ready with text domain `sangia.org`. You can find translation files in the `/languages` directory.
